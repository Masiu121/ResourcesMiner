# ResourcesMiner

## What is this game about?
In this game, you can mine resources such as coal, copper, iron, etc.
You have to upgrade your miner if you want to go deeper. There will be 5 tiers of each miner component.

### Miner components
* Drill
* Chassis
* Body
* Cooler
* Fuel tank

### Ores
* Coal ore
* Copper ore
* Iron ore
* Apatite ore
* Diamond ore
* Emerald ore
